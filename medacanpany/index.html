ゆりもちゃんはとっても臆病だけどほんとはね… / A5表FCオンデマ / 30P / 300円

趣味の歯車 / A5表FCオンデマ / 28P / 300円

千曲朝美には昔からマンガ以外にこれといった趣味も関心もなかった。

かすかに距離を感じながらも、朝美は漫画を描き続ける。
しかし――

普通の人生と私が少し夢見た人生 / A5表FCオンデマ / 44P / 500円>

内気で絵を描くことが好きな小学生の知佳は、同じく絵を描くことが好きだがクラスでは浮いた存在の沙織と友達になる。
知佳は沙織と遊ぶうちに漫画家になることを夢見るが、年相応の『普通の』少女らしさへの憧れも持っており…
同じ夢を持っているが、その熱量は違う――そんな二人の話。

発行日：2018/05/05
